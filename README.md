🎓 Campus Buddy AI

An AI-powered campus companion that helps students with academics, productivity, and campus-related information—all in one place.

🚀 Live Demo: https://preview--campus-buddy-ai-045.lovable.app/

📖 Overview

Campus Buddy AI is a modern web application designed to simplify student life by providing an intelligent AI assistant that helps with academic queries, study planning, campus information, and productivity.

Instead of searching across multiple websites or asking seniors, students can interact with a single AI assistant to get instant guidance and relevant information.

This project demonstrates the practical use of Generative AI in education with a clean, responsive, and user-friendly interface.

❗ Problem Statement

College students often struggle with:

Finding academic information quickly
Managing study schedules
Getting instant answers to doubts
Accessing campus-related resources
Switching between multiple websites and apps

These issues waste time and reduce productivity.

💡 Solution

Campus Buddy AI provides an intelligent assistant that enables students to:

Ask academic questions
Receive AI-generated responses
Access campus-related information
Improve study productivity
Use a clean and responsive interface on any device
